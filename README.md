Installation Guide:

<br>**Step: 1. Update und Upgrade**

```
$ apt-get update && apt-get upgrade -y
```

<br>**Step: 2. Install Git**

```
$ apt-get install git -y
```

<br>**Step: 3. Go to "wakeonlan_php" folder**

```
$ cd wakeonlan_php
```

<br>**Step: 4. Run install script**

```
$ chmod +x
$ bash install.sh
```
